नागरिक सेवा

ग्रामपंचायत स्तरावर देण्यात येणाऱ्या सेवा

जन्म नोंदणी

गावातील नवजात बालकाचा जन्म नोंदवण्यासाठी

सेवेची प्रक्रिया:

  • हॉस्पिटल/डॉक्टरकडून जन्म प्रमाणपत्र घ्यावे
  • ग्रामपंचायत कार्यालयात अर्ज करावा
  • तपासणी करून ७ दिवसांत प्रमाणपत्र दिले जाईल

कागदपत्रे: जन्म प्रमाणपत्र, आई-वडिलांचे आधार कार्ड

शुल्क: ₹२०

कालमर्यादा: ७ दिवस

संपर्क: सचिव, मो. ९५६१८८५०५१

फॉर्म डाउनलोड
मृत्यू नोंदणी

गावातील नागरिकांचे मृत्यू नोंदवण्यासाठी

प्रक्रिया: फॉर्म भरून → कागदपत्र → प्रमाणपत्र

कागदपत्रे: वैद्यकीय मृत्यू प्रमाणपत्र, कुटुंब ओळखपत्र

शुल्क: ₹२०

कालमर्यादा: ३ कार्यदिवस

फॉर्म डाउनलोड
विवाह नोंदणी

लग्नाचे अधिकृत नोंदणी व प्रमाणपत्र

प्रक्रिया: फॉर्म → कागदपत्र → प्रमाणपत्र

कागदपत्रे: विवाह दस्तऐवज, ओळखपत्रे

शुल्क: ₹२०

कालमर्यादा: ७-१५ दिवस

फॉर्म डाउनलोड
राहिवासी दाखला

स्थायी राहिवासीतेची पूर्तता व दाखला

प्रक्रिया: फॉर्म → कागदपत्र → प्रमाणपत्र

कागदपत्रे: आधार/रेशन कार्ड/विधुत बिल

शुल्क: ₹२५

कालमर्यादा: ३ कार्यदिवस

फॉर्म डाउनलोड
उत्पन्न दाखला

आर्थिक मदतीसाठी उत्पन्न प्रमाणपत्र

प्रक्रिया: फॉर्म → कागदपत्र → प्रमाणपत्र

कागदपत्रे: ओळखपत्र, पत्ता पुरावा

शुल्क: ₹६६

कालमर्यादा: ७-१५ दिवस

फॉर्म डाउनलोड
दारिद्र्य दाखला

गरजूंना लाभ देण्यासाठी दारिद्र्य प्रमाणपत्र

प्रक्रिया: फॉर्म → कागदपत्र → प्रमाणपत्र

कागदपत्रे: आधार, उत्पन्न पुरावे

शुल्क: मोफत

कालमर्यादा: १५-३० दिवस

फॉर्म डाउनलोड
जात दाखला

शैक्षणिक व योजना साठी जात प्रमाणपत्र

प्रक्रिया: फॉर्म → कागदपत्र → प्रमाणपत्र

कागदपत्रे: जात संदर्भ, ओळखपत्र

शुल्क: ₹१२६

कालमर्यादा: ७-१५ दिवस

फॉर्म डाउनलोड
व्यवसाय परवाना

स्थानीय व्यवसायासाठी परवाना जारी

प्रक्रिया: फॉर्म → कागदपत्र → परवाना

कागदपत्रे: व्यवसाय पत्ता, ओळखपत्र

शुल्क: फी बदलू शकते

कालमर्यादा: १५-३० दिवस

बांधकाम परवाना

प्रक्रिया: फॉर्म → नकाशा → परवाना

कागदपत्रे: डिझाईन नकाशा, मालकी कागद

शुल्क: असल्यास

कालमर्यादा: ३०-४५ दिवस

प्रक्रिया: फॉर्म → कागदपत्र → कनेक्शन

कागदपत्रे: घरचा पत्ता, ओळखपत्र

शुल्क: फॉर्मल फी
